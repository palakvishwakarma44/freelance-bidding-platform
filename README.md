# 💼 AI-Powered Freelance Bidding Platform

An advanced full-stack freelancing platform designed to connect clients and freelancers with **AI-driven project insights**, **team-based bidding**, and a modern user experience.

---

## 🚀 Key Features

### 🧠 AI Project Architect
- Integrated **Gemini 1.5 Flash (Google AI)**  
- Automatically generates **technical blueprints** for project posts  
- Helps freelancers understand requirements instantly  

---

### 👥 Squad & PoC Bidding
- Freelancers can **bid as a team (Squad System)**  
- Submit **interactive Proof of Concept (PoC)** snippets  
- Clients can preview solutions before hiring  

---

### 🎨 Premium User Experience
- Glassmorphic **Login/Register UI**  
- Responsive **split-screen layout**  
- Clean and modern dashboard design  

---

### ⚙️ Core Functionalities
- 🔐 Secure Authentication & Authorization  
- 📊 Project & Bid Management System  
- 💬 Real-time Messaging (WebSockets)  
- 🔔 Smart Notifications  
- ⭐ Reviews & Contract Handling  

---

## 🛠 Tech Stack
- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Real-time:** WebSockets  
- **AI Integration:** Gemini API  
- **Languages:** JavaScript, HTML, CSS  

---

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/palakvishwakarma44/freelance-bidding-platform.git
cd freelance-bidding-platform
```

2. Install dependencies for both client and server:
```bash
# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
```

3. Run the application:
```bash
# Start server (from /server)
npm run dev

# Start client (from /client)
npm run dev
```
