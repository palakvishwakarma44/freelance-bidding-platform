## 📦 Installation  

1. Clone the repository:
```bash
git clone https://github.com/palakvishwakarma44/freelance-bidding-platform.git
cd freelance-bidding-platform
```

2. Install dependencies:
```bash
cd server
npm install

cd ../client
npm install
```

3. Run project:
```bash
cd server
npm run dev

cd ../client
npm run dev
```
